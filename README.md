# Portfolio Website - Sapeksh Vishwakarma

A modern, professional, and fully responsive portfolio website showcasing skills, projects, education, and experience as a Full Stack Developer.

## Features

- Modern & Professional Design - Clean UI with smooth animations
- Fully Responsive - Works seamlessly on all devices (mobile, tablet, desktop)
- Light/Dark Mode Toggle - Switch between themes with persistent preference
- Smooth Animations - ScrollReveal and custom CSS transitions
- Interactive Elements - Hover effects and dynamic content
- Direct Contact Links - Email, GitHub, and LinkedIn integration
- SEO Optimized - Meta tags and semantic HTML
- Accessible - ARIA labels and semantic structure
- Fast Loading - Optimized performance with minimal dependencies

## Tech Stack

### Frontend
- **HTML5** - Semantic markup
- **CSS3** - Modern styling with Grid, Flexbox, CSS Variables
- **JavaScript** - Interactive functionality

### Libraries & Tools
- **Font Awesome** - Icon library
- **Google Fonts** - Inter typography
- **LocalStorage API** - Theme persistence
- **Intersection Observer** - Scroll animations

## Design Highlights

- **Color Palette**: 
  - **Dark Mode**: 
    - Primary: `#2C3E50` (Dark blue-gray)
    - Accent: `#3498DB` (Blue)
    - Text: `#ECF0F1` (Light gray)
  - **Light Mode**:
    - Primary: `#F8F9FA` (Light gray)
    - Accent: `#2980B9` (Blue)
    - Text: `#2C3E50` (Dark blue-gray)
- **Typography**: Inter font family
- **Layout**: CSS Grid and Flexbox
- **Animations**: Smooth transitions and hover effects
- **Components**: Modular and reusable design patterns
- **Theme Toggle**: Persistent theme preference with smooth transitions

## Sections

1. **Home** - Introduction with profile photo, role title, and social links
2. **About** - Personal information, skills overview, and professional highlights
3. **Skills** - Technical skills organized by categories (Languages, Frameworks, Databases, Tools)
4. **Education** - Academic background with timeline visualization
5. **Experience** - Professional internship details and achievements
6. **Projects** - Featured projects with descriptions and tech stacks
7. **Contact** - Direct contact links (Email, GitHub, LinkedIn)

## Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/sapeksh123/My_Portfolio.git
cd My_Portfolio
```

### 2. Open in Browser
Simply open `index.html` in your preferred browser, or use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve

# Or use VS Code Live Server extension
```

### 3. Customize Content
- Update personal information in `index.html`
- Replace profile picture in `files/Profile.png`
- Update resume in `files/` folder
- Modify colors in `styles.css` (CSS variables at the top)

## Theme Toggle

The website features a light/dark mode toggle that:
- Saves user preference in localStorage
- Applies theme consistently across all pages
- Smooth transitions between themes
- Optimized colors and shadows for both modes
- Icon changes based on current theme (sun/moon)
